Botones

Habanero incluye dos estilos principales de botones.

*El "default" el cual emplea un gradiente y tiene las acciones "hover" y "active" para que este se vea más dinámico

*El "line" el cual consta de fondo blanco y al hacer "hover" el color de fondo cambia dependiendo de la clase que se le asigne.

Además incluye diversas formas, tamaños y varios colores predeterminados los cuales se mostrarán más adelante.

<button type="button" class="button"> Default </button>

Etiquetas aplicables

Las clases diseñadas para darle un estilo a los botones también pueden ser utilizadas en otras etiqueta, tal como divs, links o inputs, aunque en el caso de los input tienen su propio estilo disponible


Div
Link 
              <div class="button">Div</div>
              <a class="button button__line">Link</a>
Tamaños

Este framework incluye 5 tamaños predeterminados de botones (pequeño, mediano, largo, extra largo y el predeterminado).

              <button type="button" class="button button__small">Small size</button>

              <button type="button" class="button button__medium">Medium size</button>
              
              <button type="button" class="button button__large">Large size</button>
              
              <button type="button" class="button button__xlarge">Extra Large size</button>
Formas

Existen formas predefinidas para los botones las cuales son:

              <button type="button" class="button button__circle">Circle</button>
              <button type="button" class="button button__square">Square</button>
              <button type="button" class="button button__trapezoid">Trapezoid</button>
