<input class="form__textObjects" type="text" name="txt1" id="txt1">
Input Box

Existe la clase form__box, la cual se debe aplicar a el conjunto de formularios que se quieran agrupar. De esta forma lo que este fuera de la caja se le aplicara un margin y un padding para distanciar los demás objetos


Default:

<div class="form__box">
              <input class="form__textObjects" type="text" 
               name="txt1" id="txt1">
              <input class="form__textObjects" type="text" 
               name="txt1" id="txt1">
</div>
              <input class="form__textObjects" type="text" 
               name="txt1" id="txt1">
