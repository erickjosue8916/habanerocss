Sistema de rejilla
Utiliza nuestra cuadricula de rejillas para crear excelentes diseños y adaptarlos a distintas pantallas, con 12 columnas.

¿Como funciona?
Utiliza una serie de contenedores, filas y columnas para tener un mejor alineamiento del contenido. Nos permite crear nuestro proyecto de manera responsive.

Tipos de adaptacion en pantalla
lg-[1-12]: toma el tamaño en pantalla grande.
md-[1-12]: toma el tamaño en pantalla mediana.
sm-[1-12]: toma el tamaño en pantalla pequeña.

Tomando en cuenta que dentro de una fila debe de ir una columna si o si, y dentro de las columnas un contenedor para tener un mejor orden de nuestras rejillas.
